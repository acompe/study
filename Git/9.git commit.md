# git commit
介绍：将暂存区中的内容提交到仓库中。
- git commit -m '提交附带信息'
    -将暂存区中的内容提交到仓库中。
- -type（scope）：subject
    - 提交信息的格式。
    - 根据该格式，在翻阅信息的时候可以很快过滤掉其余信息。
- type
    - feat
        - 新功能
    - fix
        - 修复bug
    - style
        - 修改格式
    - refactor
        - 代码重构
    - chore
        - 项目构建
- scope 
    - 模块
- subject
    - 详细描述信息
- 注意：
    - 在生成head信息的时候，大部分人会选择
    __git cz__
    工具进行head生成（
    __commitizen__
    ）。
    - 在生成
    __CHANGELOG.md__
    的时候，大部分人会用
    __changelog__
    工具，生成文档。
